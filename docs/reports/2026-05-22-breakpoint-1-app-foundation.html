Manual Review Steps

  1. Run the app on an iPhone simulator with the BillSplitz scheme.
  2. Confirm the first screen shows BillSplitz, New Split, disabled Continue Draft, and the six-step flow list.
  3. Tap New Split, then tap Next through Session Setup, Receipt Capture, Receipt Review, Split Board, Settlement, and Share.
  4. On any step, tap Start in the navigation bar, then tap Continue Draft to resume the latest step.
  5. On Share, tap Done and confirm the app returns to Start with no resumable draft.
